Well, I've been doing some digging, and I'm very impressed with
Pentadactyl!  The :js autocompletion is great.  After digging around
in that for far too long, and finally looking in the ScrapBook
extension's XUL files, I came up with these:

* :js ScrapBookBrowserOverlay.execCapture(0, null, true,
'urn:scrapbook:root')
        * This gets the current page and prompts for details (save as)
* :js ScrapBookBrowserOverlay.execCapture(0, null, false,
'ScrapBookContextPicking')
        * This gets the current page and prompts just for the folder to save
it in
* :js ScrapBookBrowserOverlay.execCapture(0, null, false,
'urn:scrapbook:root')
        * This immediately saves the current page to the root folder


All that remains is to map them to a key of your choosing.

I'd like to be able to dump a page straight into the last-used folder,
but ScrapBook doesn't provide that functionality itself, so that would
require some custom JS.  Maybe someday.  :)

Hope someone finds this useful.
